# 기능 및 프로그램 아이디어 (2026-06-11)

## pjt_manager 구현 완료 기능

| 기능 | 설명 |
|------|------|
| 인증 (로그인/회원가입) | Supabase Auth 기반, Zustand로 상태 관리 |
| 프로젝트 CRUD | 생성/수정/삭제/아카이브, 색상 커스텀 |
| 칸반 보드 | 컬럼 추가/삭제/색상 설정, 태스크 드래그앤드롭 |
| 태스크 CRUD | 제목/설명/우선순위/마감일/태그/메모/타입(태스크·미팅) |
| 체크리스트 | 태스크 내 하위 항목 추가/완료 처리 |
| 전역 검색 | 사이드바에서 모든 프로젝트 태스크 통합 검색 (⌘K) |
| 프로젝트 정렬 | 대시보드 — 최신순/수정순/진행률순/이름순 |
| 탭 타이틀 배지 | 기한초과 수 `(N) My PM` 표시 |
| 전체 현황 | 기한초과/오늘마감/긴급 필터, 이슈·기록 뷰, 우측 편집 패널 |
| 주간 리포트 | 생성/완료/기한초과 요약, 주차 이동, 프로젝트별 현황 |
| 이슈 & 기록 | 프로젝트별 이슈(open/closed) + 노트 관리 |
| 캘린더 | 월별 태스크·이벤트 표시, 이벤트 CRUD |
| 오늘 할 일 | 오늘 마감 태스크 + 직접 등록한 오늘 할 일 목록 |
| 사이드바 커스텀 | 메뉴 순서 변경, 프로젝트 바로가기 표시/숨김/순서 편집 |
| 다크모드 | 설정 페이지에서 라이트/다크/시스템 전환 |
| 로그인 후 즉시 데이터 로드 | `enabled: !!user` + `invalidateQueries()` 로 캐시 문제 수정 |

## pjt_manager 추가 기능 아이디어

### 통계 / 분석 대시보드

> 현재는 숫자 카운트 수준만 존재 (전체 현황 필터 뱃지, 주간 리포트 카드 3개)
> 차트 라이브러리: recharts 추천 (Next.js 궁합 좋음, 번들 작음)

| 기능 | 설명 | 난이도 |
|------|------|--------|
| 프로젝트 진행률 바 차트 | 전체/완료 태스크 비율 시각화 | 쉬움 |
| 우선순위 분포 도넛차트 | 긴급/높음/보통/낮음 비율 | 쉬움 |
| 주간 완료 추이 꺾은선 | 최근 4~8주 완료 수 트렌드 | 쉬움 |
| 컬럼별 WIP 현황 | 각 컬럼에 태스크가 몇 개 쌓여있는지 | 쉬움 |
| 태그 빈도 분석 | 자주 쓰는 태그 클라우드/차트 | 쉬움 |
| 월별 생성 vs 완료 비교 | 업무량 추이 파악 | 중간 |
| 태스크 평균 처리 기간 | 생성일~완료일 기준 소요시간 분포 | 중간 |

---

### 반응형 / 모바일 대응

| 기능 | 설명 |
|------|------|
| 사이드바 → 하단 탭바 전환 | 모바일에서 `w-56` 고정 사이드바 숨기고 하단 고정 네비게이션으로 전환 |
| 칸반 세로 스크롤 | 모바일에서 컬럼을 수직으로 쌓아 스와이프로 탐색 |
| 디테일 패널 풀스크린 | `w-[540px]` 고정 패널을 모바일에서 풀스크린 오버레이로 전환 |
| 터치 UX 최적화 | 버튼 탭 영역 확대, 드래그앤드롭 터치 지원 |
| Tailwind 반응형 적용 | `sm:/md:/lg:` 클래스 전체 페이지에 적용 |

> 현재 `sm:/md:/lg:` 사용이 사실상 전무 (4개 파일에 12군데뿐) — 전체 개편 필요

### 미구현 — 쉬움

| 기능 | 설명 |
|------|------|
| 컬럼 색상 → 카드 배경 연동 | 컬럼 색상은 있지만 카드 배경색에 미반영 |
| 태스크 일괄 선택/이동 | 체크박스로 여러 카드 선택 후 일괄 컬럼이동/삭제 |

### 미구현 — 중간

| 기능 | 설명 |
|------|------|
| 태스크 댓글/활동 로그 | 카드에 메모 남기기 + 변경 이력 (DB 테이블 신규 필요) |
| 반복 태스크 | 매주 월요일 자동 생성 등 규칙 설정 |
| 파일 첨부 | Supabase Storage 연동, 카드에 파일 첨부 |
| 칸반 → 리스트 뷰 전환 | 테이블 형태로 보기 (엑셀처럼) |

### 미구현 — 고도화

| 기능 | 설명 |
|------|------|
| 팀원 초대 / 공유 | 프로젝트 단위 멤버 초대, 태스크 담당자 지정 (단일 user_id 구조 전면 개편 필요) |
| Slack / 카카오 알림 | 마감 D-1 자동 알림 Webhook |
| AI 태스크 분해 | 큰 태스크 입력 시 하위 체크리스트 자동 생성 (Claude API) |

## 새로운 프로그램 아이디어

### 1. 견적/발주 관리 시스템 ★ 추천 1순위
> POSCO E&C, KOGAS 등 거래처 관리 + 견적서 생성 + 발주 상태 추적
- 거래처 DB, 품목 DB
- 견적서 PDF 자동 생성
- 발주 → 납품 → 정산 단계 관리
- pjt_manager 프로젝트와 연동 가능

### 2. 거래처 CRM ★ 추천 2순위
> 고객사별 연락처, 미팅 이력, 계약 현황 통합 관리

- 연락처 + 미팅 기록
- 계약 갱신 알림
- pjt_manager 캘린더와 연동 가능

### 3. 현장 점검 체크리스트 앱
> 소방/설비 현장 점검 항목을 모바일로 체크

- QR코드로 현장 진입
- 점검 항목 체크 + 사진 첨부
- 결과 리포트 PDF 출력

### 4. 내부 문서/도면 관리
> 프로젝트별 도면, 시방서, 승인서 파일 관리

- 버전 관리 (v1, v2...)
- 검색 + 태그
- 다운로드 이력

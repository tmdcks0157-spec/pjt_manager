## pjt_manager 추가 기능 아이디어

### 즉시 구현 가능 (쉬움)

| 기능 | 설명 |
|------|------|
| 태스크 마감일 알림 배지 | 브라우저 탭 타이틀에 `(3) My PM` 처럼 기한초과 수 표시 |
| 프로젝트 정렬/필터 | 대시보드에서 진행률순, 최근수정순 정렬 |
| 태스크 검색 (전역) | 모든 프로젝트 통합 검색 — 현재는 프로젝트 내 검색만 가능 |
| 컬럼별 색상 커스텀 | 이미 있지만 카드 배경색도 컬럼 색상 연동 |
| 태스크 일괄 선택/이동 | 체크박스로 여러 카드 선택 후 일괄 컬럼이동/삭제 |

### 중간 난이도

| 기능 | 설명 |
|------|------|
| 주간 리포트 | 이번 주 완료/생성/지연 태스크 요약 페이지 |
| 태스크 댓글/활동 로그 | 카드에 메모 남기기 + 변경 이력 |
| 반복 태스크 | 매주 월요일 자동 생성 등 규칙 설정 |
| 파일 첨부 | Supabase Storage 연동, 카드에 파일 첨부 |
| 칸반 → 리스트 뷰 전환 | 테이블 형태로 보기 (엑셀처럼) |

### 고도화

| 기능 | 설명 |
|------|------|
| 팀원 초대 / 공유 | 프로젝트 단위 멤버 초대, 태스크 담당자 지정 |
| Slack / 카카오 알림 | 마감 D-1 자동 알림 Webhook |
| AI 태스크 분해 | 큰 태스크 입력 시 하위 체크리스트 자동 생성 (Claude API) |

## 새로운 프로그램 아이디어

### 1. 견적/발주 관리 시스템 ★ 추천 1순위
> POSCO E&C, KOGAS 등 거래처 관리 + 견적서 생성 + 발주 상태 추적

- 거래처 DB, 품목 DB
- 견적서 PDF 자동 생성
- 발주 → 납품 → 정산 단계 관리
- pjt_manager 프로젝트와 연동 가능

### 2. 거래처 CRM ★ 추천 2순위
> 고객사별 연락처, 미팅 이력, 계약 현황 통합 관리

- 연락처 + 미팅 기록
- 계약 갱신 알림
- pjt_manager 캘린더와 연동 가능

### 3. 현장 점검 체크리스트 앱
> 소방/설비 현장 점검 항목을 모바일로 체크

- QR코드로 현장 진입
- 점검 항목 체크 + 사진 첨부
- 결과 리포트 PDF 출력

### 4. 내부 문서/도면 관리
> 프로젝트별 도면, 시방서, 승인서 파일 관리

- 버전 관리 (v1, v2...)
- 검색 + 태그
- 다운로드 이력

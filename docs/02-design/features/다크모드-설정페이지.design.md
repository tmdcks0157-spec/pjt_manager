# Design: 다크모드 + 설정 페이지

> Plan: `docs/01-plan/features/다크모드-설정페이지.plan.md`

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 앱이 라이트 모드만 지원해 어두운 환경에서 눈 피로가 크고, 사용자 환경설정을 바꿀 방법이 없다 |
| **Solution** | `/settings` 페이지 신설 + Tailwind v4 `dark:` class 전략으로 다크모드 구현, `localStorage`에 저장 |
| **Function UX Effect** | 설정 한 곳에서 테마·시작 페이지 변경, 새로고침 후에도 유지, FOUC(깜빡임) 없는 다크 UI |
| **Core Value** | "내 방식대로 쓰는 개인 도구" — 꼭 필요한 환경 커스터마이징만 |

### 2.2 `src/app/layout.tsx` — FOUC 방지 인라인 스크립트

루트 Layout은 Server Component이므로 `<head>`에 직접 `<script>` 삽입 가능.
React 하이드레이션 전에 실행되므로 깜빡임 없이 테마 적용.

```tsx
// src/app/layout.tsx
import type { Metadata } from 'next'
import { Geist } from 'next/font/google'
import './globals.css'
import { QueryProvider } from '@/providers/QueryProvider'
import { ThemeProvider } from '@/providers/ThemeProvider'

const geistSans = Geist({ variable: '--font-geist-sans', subsets: ['latin'] })

export const metadata: Metadata = {
  title: 'My Project Manager',
  description: '나만의 프로젝트 관리 도구',
}

// FOUC 방지: 하이드레이션 전에 실행되는 인라인 스크립트
const themeScript = `
  (function() {
    try {
      var t = localStorage.getItem('theme');
      if (t === 'dark' || (!t && window.matchMedia('(prefers-color-scheme: dark)').matches && t === 'system')) {
        document.documentElement.classList.add('dark');
      }
      if (t === 'dark') document.documentElement.classList.add('dark');
    } catch(e) {}
  })()
`

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="ko" className={`${geistSans.variable} h-full antialiased`}>
      <head>
        <script dangerouslySetInnerHTML={{ __html: themeScript }} />
      </head>
      <body className="min-h-full bg-gray-50 dark:bg-gray-900 text-gray-900 dark:text-gray-100">
        <QueryProvider>
          <ThemeProvider>
            {children}
          </ThemeProvider>
        </QueryProvider>
      </body>
    </html>
  )
}
```

**FOUC 방지 로직 상세:**
```
1. localStorage['theme'] === 'dark'  → html.dark 추가
2. localStorage['theme'] === 'system' → OS 다크모드 감지 후 추가
3. localStorage['theme'] === 'light' 또는 없음 → 아무것도 안 함
```

### 2.3 `src/providers/ThemeProvider.tsx`

```tsx
'use client'

import { createContext, useContext, useEffect, useState, ReactNode } from 'react'

type Theme = 'light' | 'dark' | 'system'

interface ThemeContextValue {
  theme: Theme
  resolvedTheme: 'light' | 'dark'  // 'system' 선택 시 실제 값
  setTheme: (t: Theme) => void
}

const ThemeContext = createContext<ThemeContextValue>({
  theme: 'light',
  resolvedTheme: 'light',
  setTheme: () => {},
})

export function ThemeProvider({ children }: { children: ReactNode }) {
  const [theme, setThemeState] = useState<Theme>('light')
  const [resolvedTheme, setResolvedTheme] = useState<'light' | 'dark'>('light')

  // 마운트 시 localStorage에서 초기값 읽기
  useEffect(() => {
    const saved = (localStorage.getItem('theme') as Theme) ?? 'light'
    setThemeState(saved)
    applyTheme(saved)
  }, [])

  // OS 다크모드 변경 감지 ('system' 모드일 때만)
  useEffect(() => {
    const mq = window.matchMedia('(prefers-color-scheme: dark)')
    const handler = () => {
      if (theme === 'system') applyTheme('system')
    }
    mq.addEventListener('change', handler)
    return () => mq.removeEventListener('change', handler)
  }, [theme])

  function applyTheme(t: Theme) {
    const isDark =
      t === 'dark' ||
      (t === 'system' && window.matchMedia('(prefers-color-scheme: dark)').matches)

    document.documentElement.classList.toggle('dark', isDark)
    setResolvedTheme(isDark ? 'dark' : 'light')
  }

  function setTheme(t: Theme) {
    localStorage.setItem('theme', t)
    setThemeState(t)
    applyTheme(t)
  }

  return (
    <ThemeContext.Provider value={{ theme, resolvedTheme, setTheme }}>
      {children}
    </ThemeContext.Provider>
  )
}

export function useTheme() {
  return useContext(ThemeContext)
}
```

### 2.5 `src/app/(main)/settings/page.tsx` — 설정 페이지

```tsx
'use client'

import { useTheme } from '@/providers/ThemeProvider'
import { useState, useEffect } from 'react'
import { useAuthStore } from '@/stores/auth-store'

type StartPage = '/today' | '/dashboard' | '/overview' | '/calendar' | '/report'

const START_PAGE_OPTIONS: { value: StartPage; label: string }[] = [
  { value: '/today',     label: '오늘' },
  { value: '/dashboard', label: '프로젝트' },
  { value: '/overview',  label: '전체 현황' },
  { value: '/calendar',  label: '캘린더' },
  { value: '/report',    label: '주간 리포트' },
]

const THEME_OPTIONS = [
  { value: 'light', label: '라이트', icon: '☀️' },
  { value: 'dark',  label: '다크',   icon: '🌙' },
  { value: 'system',label: '시스템', icon: '💻' },
] as const

export default function SettingsPage() {
  const { theme, setTheme } = useTheme()
  const { user } = useAuthStore()
  const [startPage, setStartPage] = useState<StartPage>('/today')

  useEffect(() => {
    const saved = localStorage.getItem('start-page') as StartPage
    if (saved) setStartPage(saved)
  }, [])

  function handleStartPageChange(value: StartPage) {
    setStartPage(value)
    localStorage.setItem('start-page', value)
  }

  return (
    <div className="p-8 max-w-2xl">
      <h1 className="text-xl font-bold text-gray-900 dark:text-gray-100 mb-8">설정</h1>

      {/* 계정 정보 (읽기 전용) */}
      <section className="mb-8">
        <h2 className="text-sm font-semibold text-gray-500 dark:text-gray-400 uppercase tracking-wider mb-3">계정</h2>
        <div className="bg-white dark:bg-gray-800 rounded-xl border border-gray-200 dark:border-gray-700 p-4">
          <p className="text-xs text-gray-500 dark:text-gray-400 mb-1">이메일</p>
          <p className="text-sm text-gray-800 dark:text-gray-200">{user?.email}</p>
        </div>
      </section>

      {/* 테마 */}
      <section className="mb-8">
        <h2 className="text-sm font-semibold text-gray-500 dark:text-gray-400 uppercase tracking-wider mb-3">테마</h2>
        <div className="flex gap-3">
          {THEME_OPTIONS.map(opt => (
            <button
              key={opt.value}
              onClick={() => setTheme(opt.value)}
              className={`flex-1 flex flex-col items-center gap-2 py-4 rounded-xl border-2 transition-all
                ${theme === opt.value
                  ? 'border-blue-500 bg-blue-50 dark:bg-blue-950'
                  : 'border-gray-200 dark:border-gray-700 bg-white dark:bg-gray-800 hover:border-gray-300 dark:hover:border-gray-600'
                }`}
            >
              <span className="text-2xl">{opt.icon}</span>
              <span className={`text-sm font-medium
                ${theme === opt.value
                  ? 'text-blue-600 dark:text-blue-400'
                  : 'text-gray-700 dark:text-gray-300'
                }`}>
                {opt.label}
              </span>
            </button>
          ))}
        </div>
      </section>

      {/* 시작 페이지 */}
      <section className="mb-8">
        <h2 className="text-sm font-semibold text-gray-500 dark:text-gray-400 uppercase tracking-wider mb-3">로그인 후 시작 페이지</h2>
        <div className="bg-white dark:bg-gray-800 rounded-xl border border-gray-200 dark:border-gray-700 overflow-hidden">
          {START_PAGE_OPTIONS.map((opt, i) => (
            <button
              key={opt.value}
              onClick={() => handleStartPageChange(opt.value)}
              className={`w-full flex items-center justify-between px-4 py-3 text-sm transition-colors
                ${i > 0 ? 'border-t border-gray-100 dark:border-gray-700' : ''}
                ${startPage === opt.value
                  ? 'text-blue-600 dark:text-blue-400 bg-blue-50 dark:bg-blue-950'
                  : 'text-gray-700 dark:text-gray-300 hover:bg-gray-50 dark:hover:bg-gray-750'
                }`}
            >
              <span>{opt.label}</span>
              {startPage === opt.value && (
                <span className="w-2 h-2 rounded-full bg-blue-500" />
              )}
            </button>
          ))}
        </div>
      </section>
    </div>
  )
}
```

| `text-gray-700` | `dark:text-gray-300` | 본문 텍스트 |
| `text-gray-600` | `dark:text-gray-400` | 보조 텍스트 |
| `text-gray-500` | `dark:text-gray-500` | 플레이스홀더 (그대로) |

## 6. 완성 기준 (Acceptance Criteria)

| 기준 | 검증 방법 |
|------|-----------|
| 다크모드 전환 즉각 반응 | 설정 페이지에서 클릭 즉시 UI 변경 확인 |
| 새로고침 후 테마 유지 | 다크 설정 후 F5 새로고침 → 깜빡임 없이 다크 유지 |
| 시스템 테마 연동 | OS 다크모드 켜고 'system' 선택 → 자동 적용 |
| 시작 페이지 설정 저장 | 시작 페이지 변경 후 로그아웃→로그인 → 설정한 페이지로 이동 |
| 라이트 모드 회귀 없음 | 라이트 모드에서 기존 UI 외관 동일 유지 |
| 사이드바에서 설정 접근 | 사이드바 하단 '설정' 링크 클릭 → /settings 이동 |

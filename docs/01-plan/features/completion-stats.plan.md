# Plan: completion-stats — 완료 통계 시각화

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 현재 report 페이지에 주간 완료 숫자는 있지만 차트가 없어 추세·패턴을 한눈에 파악하기 어렵다 |
| **Solution** | 기존 report 페이지에 주간·월간 완료 바 차트 추가 — 데이터는 이미 있으므로 시각화 레이어만 얹기 |
| **Function UX Effect** | 이번 주 어느 요일에 많이 완료했는지, 이번 달 어느 주가 생산적이었는지 막대 차트로 즉시 파악 |
| **Core Value** | 완료 기록이 동기부여 데이터로 전환 — "나는 월요일에 가장 많이 끝낸다" 같은 인사이트 제공 |

---

## 1. 현황 분석

### 기존 재활용 가능한 것

| 항목 | 내용 |
|------|------|
| `src/app/(main)/report/page.tsx` | 이미 주간 완료 태스크 데이터 쿼리 중 |
| `useWeeklyReport` 훅 | 주간 완료/진행 데이터 있음 |
| `tasks` 테이블 | `created_at`, `status`, `project_id` 활용 가능 |

### 새로 필요한 것

| 항목 | 설명 |
|------|------|
| 차트 라이브러리 | `recharts` (Next.js 친화적, 번들 경량) 추천 |
| 월간 집계 쿼리 | 현재 주간만 있음, 월간 추가 필요 |
| 차트 컴포넌트 | `WeeklyBarChart`, `MonthlyBarChart` |

---

## 2. 기능 범위

### 2.1 주간 완료 차트

| 항목 | 상세 |
|------|------|
| 형태 | 요일별 바 차트 (월~일) |
| 데이터 | 해당 요일에 완료 컬럼으로 이동된 태스크 수 |
| 위치 | report 페이지 상단 (기존 숫자 요약 아래) |

### 2.2 월간 완료 차트

| 항목 | 상세 |
|------|------|
| 형태 | 주차별 바 차트 (1주~5주) |
| 데이터 | 해당 주에 완료된 태스크 수 |
| 전환 | 주간/월간 탭 전환 또는 토글 |

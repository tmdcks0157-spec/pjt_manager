## 기술 메모
- `.env.local`은 `.gitignore`에 포함되어 있어 새 기기마다 수동 생성 필요
- Supabase 부분 select 시 TypeScript 타입을 `Pick<T, ...>` 또는 `as` 캐스팅으로 명시해야 Vercel 빌드 통과
- 완료 태스크 판별: `columns` 테이블에서 `name === '완료'`인 컬럼 ID를 기준으로 집계
- 다른 프로젝트로 이동 시 대상 프로젝트의 첫 컬럼을 on-demand로 조회 (`order` 기준 `limit 1`)
- 카드 펼치기 상태는 React 로컬 state (`useState`) — 새로고침 시 초기화 (의도적)
- PostgREST `.neq('col', value)` 는 `col IS NULL` 행도 제외함 — `archived` 포함 여부 주의
- Soft delete 패턴: `deleted_at` timestamp — `null` = 활성, 값 있으면 삭제된 것으로 처리

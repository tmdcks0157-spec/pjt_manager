## 1. CRM 기능 신규 구현

### DB 변경 (Supabase)
- `companies` 테이블 신규 생성 (회사명, 업종, 웹사이트, 전화, 메모, 태그)
- `contacts` 테이블 신규 생성 (이름, 이메일, 전화번호 배열, 직함, 회사 연결, 태그, 메모)
- `activities` 테이블 신규 생성 (통화/이메일/미팅/메모 활동 타임라인)
- `tasks` 테이블에 `contact_id` 컬럼 추가
- `posts` 테이블에 `contact_id`, `recorded_at` 컬럼 추가
- 전체 RLS 정책 적용

### 신규 파일
- `src/hooks/useCRM.ts` — 연락처/회사 CRUD React Query 훅
- `src/hooks/useActivities.ts` — 활동 로그 CRUD 훅
- `src/components/crm/ContactForm.tsx` — 연락처 등록/수정 모달
- `src/components/crm/CompanyForm.tsx` — 회사 등록/수정 모달
- `src/components/crm/ActivityTimeline.tsx` — 활동 타임라인 컴포넌트
- `src/components/crm/ActivityForm.tsx` — 활동 기록 인라인 폼
- `src/app/(main)/crm/page.tsx` — 연락처 목록 페이지 ([사람]/[회사] 탭)
- `src/app/(main)/crm/contacts/[id]/page.tsx` — 연락처 상세 페이지

### 수정 파일
- `src/app/(main)/layout.tsx` — 사이드바에 연락처(CRM) 메뉴 추가
- `src/types/index.ts` — Company, Contact, Activity, ActivityType 타입 추가

## 3. CRM UI 개선

### 다음 액션 기능 제거
- ContactForm에서 다음 액션(날짜/메모) 필드 제거
- 연락처 상세 헤더 배너 제거
- 칸반 태스크 카드로 다음 액션 관리하는 방식으로 전환
- `useOverdueNextActions` 훅 제거

### 전화번호 복수 입력
- 기존 단일 `phone` 컬럼 → `phones text[]` 배열 컬럼으로 변경
- ContactForm에서 + 번호 추가 / X 삭제 버튼으로 여러 번호 등록 가능

### 연락처 카드 정보 표시
- 연락처 카드에 이메일 클릭 → `mailto:` 링크
- 연락처 카드에 전화번호 클릭 → `tel:` 링크
- 연락처 카드에 활성 태스크 수 뱃지 표시 (완료/보관/삭제 제외)

### Today 페이지 "연락할 것" 섹션 개선
- 기존 next_action_date 기반 → 연락처 연결된 오늘/초과 마감 태스크 기반으로 변경
- 연락처별로 그룹핑하여 표시

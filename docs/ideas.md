### 반응형 / 모바일 대응 💡 보류

같은 Next.js 코드베이스, Tailwind 반응형 클래스 추가 방식  
태블릿 먼저(md: 768px) → 모바일 단계적 접근 권장

| 기능 | 설명 |
|------|------|
| 사이드바 → 하단 탭바 | 모바일 `w-56` 고정 사이드바 → 하단 고정 네비게이션 |
| 칸반 세로 스크롤 | 컬럼 수직 배치 + 스와이프 |
| 디테일 패널 풀스크린 | 모바일에서 `w-[540px]` 패널 → 풀스크린 오버레이 |

> 현재 `sm:/md:/lg:` 사용 극히 제한적 — 전체 적용 필요

### M365 연동 💡 보류

사용자 환경: **M365 Personal 구독 중 (업무=Windows / 모바일=iOS)**  
Microsoft Graph API + Azure App Registration 1회 셋업으로 전체 커버

| 우선순위 | 기능 |
|----------|------|
| 1순위 | Outlook Calendar 양방향 싱크 |
| 2순위 | OneDrive 파일 첨부 (태스크 파일 → OneDrive `/ProjectManager/`) |
| 3순위 | 연락처 → Microsoft Graph `POST /me/contacts` → iOS Outlook 싱크 |
| 참고 | 알림: Teams 대신 Outlook 이메일 (Teams Webhook은 Personal 미지원) |
| 참고 | Outlook 메일 → 태스크 자동 생성 (Mail.Read 권한) |

## 신규 앱 아이디어

### 생산성 앱

| 앱 | 설명 | 추천도 |
|----|------|--------|
| **타임 트래커** | 작업별 시간 기록 + Pomodoro 타이머, my-project-manager 태스크 연동 가능 | ⭐⭐⭐ |
| **주간 계획+회고** | 매주 월요일 계획 → 금요일 회고 루틴, GTD 방식 | ⭐⭐ |
| **OKR 목표 관리** | 분기/연간 목표 → 핵심 결과 → 주간 태스크 연결, 달성률 시각화 | ⭐⭐ |
| **에너지 트래커** | 하루 3~4회 에너지/집중도 1~5점 체크인, 주간 패턴 분석 | ⭐ |
